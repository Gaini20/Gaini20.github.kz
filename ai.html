Кіріспе

Жасанды интеллект деген не? Қолдану салалары және тарихы

Жасанды интеллект (ЖИ) — бұл компьютерлер мен машиналарға адамның ақыл-ойына тән қабілеттерді (ойлау, үйрену, шешім қабылдау, тіл түсіну, көру, қозғалу т.б.) үйретуге бағытталған ғылым мен технология саласы. Қарапайым тілмен айтқанда, ЖИ — адамның ақыл-ой әрекетін еліктейтін және өздігінен шешім қабылдай алатын бағдарламалық жүйе.

Жасанды интеллекттің негізгі сипаттамалары:

  • Ойлау: деректерді талдау, салыстыру, логикалық тұжырым жасау;
  • Үйрену: тәжірибе арқылы өз жұмысын жақсарту (Machine Learning);
  • Анализ жасау: үлкен көлемдегі деректерді өңдеу және мағыналы нәтиже шығару;
  • Автоматтандыру: адамның қатысуынсыз шешім қабылдау және әрекет ету;
  • Болжам жасау: өткен мәліметтерге сүйене отырып болашақты болжау.

    1. Медицина саласы. Медицина ЖИ ең тиімді қолданылатын салалардың бірі. ЖИ алгоритмдері ауруларды ерте анықтауға, диагноз қоюға, емдеу жолдарын ұсынуға көмектеседі. Медициналық суреттерді (рентген, томография) талдау арқылы ЖИ дәрігердің жұмысын жеңілдетіп, қатерлі ісік сияқты ауруларды бастапқы сатысында анықтауға мүмкіндік береді. IBM Watson Health секілді жүйелер науқастың бүкіл тарихын сараптап, нақты ем тәсілдерін ұсынады.
    2. Білім беру. Білім беру саласында ЖИ оқушылардың жеке ерекшеліктерін ескере отырып, бейімделген оқу бағдарламаларын құруға мүмкіндік береді. Онлайн платформаларда (мысалы, Coursera, Khan Academy) ЖИ арқылы оқушылардың үлгерімі мен қателері сараланып, жеке ұсыныстар беріледі. Сонымен қатар, ЖИ чат-боттар оқушылардың сұрақтарына жылдам жауап беріп, мұғалімнің уақытын үнемдейді.
    3. Көлік және логистика. Автономды көліктер мен смарт логистика – ЖИ-дің бұл саладағы жетістіктерінің жарқын мысалы. Tesla және Waymo компаниялары жүргізушісіз көліктер жасап, жол қауіпсіздігін арттыруда. Ал логистикалық жүйелерде ЖИ жүк тасымалдау бағыттарын оңтайландырып, уақыт пен шығынды азайтуға көмектеседі.
    4. Қаржы саласы. ЖИ банк және сақтандыру жүйелерінде тәуекелді талдау, алаяқтықты анықтау, несие қабілеттілігін болжау үшін пайдаланылады. Kaspi, Halyk Bank сияқты банктер ЖИ негізінде қолданушы әрекеттерін талдап, жеке ұсыныстар береді. Қаржы нарығында ЖИ бағаның өзгерісін болжауға да көмектеседі.
    5. Өнеркәсіп және өндіріс. Өндірісте ЖИ роботтық жүйелерді басқару, өндіріс ақауларын анықтау, сапаны автоматты тексеру үшін қолданылады. Мұндай технологиялар өнімнің сапасын арттырып, жұмысшының қауіпсіздігін қамтамасыз етеді. Өндірістегі автоматтандыру – ЖИ арқылы адам жұмысын жеңілдету мен тиімділікті арттыру жолы.

    Машиналық оқыту (Supervised Learning)

    Машиналық оқыту (Supervised Learning) — бұл деректермен жұмыс істеп, болашақта болжамдар жасау үшін қолданылатын әдіс. Бұл әдісте жүйе белгілі бір деректер жиынтығының негізінде үйренеді және жаңа мәліметтермен жұмыс істегенде оны дұрыс өңдейді.

    Машиналық оқыту қалай жұмыс істейді?

    Машиналық оқыту кезінде біз жүйеге бірнеше мысалдар береміз. Әрбір мысалдың дұрыс жауабы (парықталған деректер) беріледі. Жүйе осы мысалдардан үйренеді және болашақта жаңа деректермен жұмыс істегенде болжамдар жасайды.

    Мысалы: Біз жазған мәтіндерге қатысты жүйе оқыған сайын олардың тақырыбын болжауға тырысады немесе адам суреттерін ұсынғанда ол суреттің ішіндегі объектілерді тануға үйренеді.

    Жасанды интеллекттің бұл түрі классификация, регрессия және тағы басқа тапсырмаларды шешуге пайдаланылады.

    Машиналық оқытудың түрлері:

  • Бақыланатын оқыту (Supervised Learning): Бұл оқыту әдісінде біз жүйеге берілген деректерді белгілейміз (мәтіндер, суреттер, т.б.), және жүйе осы деректерден үйренеді.
  • Бақыланбайтын оқыту (Unsupervised Learning): Мұнда жүйе деректердің құрылымын өздігінен анықтайды, мысалы, кластерлеу.
  • Күшейтілген оқыту (Reinforcement Learning): Бұл әдіс агенттің әрекеттері үшін марапат немесе жазаны қолдануға негізделген. Агент әр қадамнан кейін ортадан кері байланыс алады.

    • Google Teachable Machine қолдану

    Google Teachable Machine — бұл өз моделіңізді жасауға мүмкіндік беретін онлайн құрал. Машиналық оқыту арқылы жүйеге суреттерді тануды үйретуге болады.

    Тапсырма: Teachable Machine платформасында суреттерді тану моделін жасаңыз. Мысалы, әртүрлі жануарлардың суреттерін қосып, оларды классификациялауға үйретіңіз.

    Қадамдар:

    1. Teachable Machine платформасына өтіңіз.
    2. «Объектілерді тану» модельін таңдаңыз.
    3. Суреттерді қосыңыз (мысалы, иттер мен мысықтар). Суреттердің санын көбейтіңіз, өйткені көп суреттер қосқан сайын, жүйе жақсы үйренеді.
    4. Модельді оқытып, оны тестілеңіз. Жүйе жаңа суретті дұрыс анықтай ала ма, тексеріңіз.
    5. Модельді сынап көріңіз, нәтижелерді талқылаңыз.

    Microsoft Azure Machine Learning қолдану

    Microsoft Azure Machine Learning — бұл үлкен деректермен жұмыс істеп, машиналық оқыту моделін жасауға мүмкіндік беретін платформа.

    Тапсырма: Microsoft Azure ML платформасын пайдаланып, берілген деректер бойынша модель жасаңыз. Мысалы, көлік жолдарының қауіпсіздігін талдайтын жүйе жасаңыз.

    Қадамдар:

    1. Microsoft Azure Machine Learning платформасына өтіңіз.
    2. Жаңа проектіні бастаңыз және деректер жиынтығын жүктеп, алдын ала өңдеуді жасаңыз.
    3. Машиналық оқыту моделін таңдаңыз (класификация, регрессия, т.б.).
    4. Модельді оқытып, нәтижелерін бағалаңыз.

    Тапсырма: OpenAI GPT-3 платформасында сұрақ қойып, оған жауап алыңыз.

    Қадамдар:

    1. OpenAI GPT-3 платформасына өтіп, тіркеліңіз немесе логин жасаңыз.
    2. Сұрақ қойыңыз, мысалы: "Жасанды интеллекттің болашағы қандай?"
    3. Жауап алыңыз және жүйенің берген ақпаратын талдаңыз.

    Деректер және алгоритмдер

    Табиғи тілдерді өңдеу (NLP) — бұл компьютерлерге адам тілін түсіну және өңдеу қабілетін беру үшін жасалған саласы. Бұл сала машиналық оқыту және лингвистика әдістерін біріктіріп, компьютерлердің мәтіндермен жұмыс істеуін жақсартады. NLP құралдары мәтіндердің мәнін түсініп, олардан маңызды ақпарат алуға мүмкіндік береді.

    NLP-дің негізгі қолданылуы:

  • Тексттің анализі: Тексттен маңызды мәліметтерді, тақырыптарды немесе эмоцияларды шығару.
  • Эмоциялық анализ: Тексттегі эмоцияларды анықтау (позитивті, негативті, нейтралды).
  • Мәтіннен ақпарат алу: Автоматтандырылған сұрақтар мен жауаптар жүйелерін жасау, мәтіндерден маңызды мәліметтерді шығару.

    • IBM Watson-ға тіркелу және кіру туралы нұсқаулық

    Қадам 1: IBM Watson платформасына өту

    1. Браузеріңізді ашып, мына сілтеме арқылы IBM Watson платформасына өтіңіз: IBM Watson NLU

    Қадам 2: Тіркелу немесе кіру

    Егер сізде IBM аккаунты жоқ болса, тіркелу қажет:

    1. Платформаға кірген соң, экранның жоғарғы оң жақ бұрышында "Sign up" (Тіркелу) батырмасын басыңыз.
    2. Электрондық поштаңызды, құпия сөзді және басқа да қажетті мәліметтерді енгізіңіз.
    3. Тіркелу аяқталған соң, тіркелген электронды поштаңызға жіберілген растау сілтемесін басу арқылы тіркеуді аяқтаңыз.

    Егер сізде IBM аккаунты болса, онда тек "Sign In" (Кіру) батырмасын басып, өзіңіздің логин мен құпия сөзіңізді енгізіңіз.

    Қадам 3: IBM Watson NLU пайдалану

    1. Тіркелген соң немесе кірген соң, IBM Watson NLU қызметін таңдау үшін платформада "Natural Language Understanding" бөлімін табыңыз.

    2. IBM Watson NLU бөлімін ашып, мәтіндермен жұмыс бастауға болады.

    Қадам 4: Мәтін талдау және эмоцияларды анықтау

    1. Мәтіндерді енгізіп, оларды талдауды бастау үшін оңай интерфейсті пайдаланыңыз.

    2. Мәтін енгізген соң, платформа эмоциялар, кілт сөздер, тақырыптар сияқты талдау нәтижелерін көрсетеді.

    Қадам 5: Нәтижелермен танысу

    1. Платформа автоматты түрде мәтіндеріңіздегі эмоцияларды талдайды (позитивті, негативті немесе нейтралды).

    2. Тақырыптар мен кілт сөздер де анықталады, бұл мәтіндердің мәнін жақсы түсінуге көмектеседі.

    Қосымша кеңестер:

  • Егер IBM Watson қызметін тегін пайдаланғыңыз келсе, тіркелген соң белгілі бір көлемде тегін сұраулар жасай аласыз. Кейбір қосымша мүмкіндіктер үшін төлем жасау қажет болуы мүмкін.
  • Егер жүйе жұмысы немесе нәтижелер бойынша сұрақтар туындаса, IBM қолдау қызметіне жүгінуге болады.

    • Практикалық тапсырма:

    IBM Watson Natural Language Understanding платформасын пайдалана отырып, мәтіннің эмоцияларын талдаңыз. Мәтіндерді енгізіп, олардың эмоциялық талдауын жасау. Платформада негізгі эмоцияларды (позитивті, негативті, нейтралды) анықтау.

    Қадамдар:

    1. IBM Watson NLU платформасына өтіңіз.
    2. Мәтінді енгізіңіз (мысалы, «Мен Python-да бағдарламалауды жақсы көремін. Бұл өте қуатты және әмбебап тіл.»).
    3. Мәтін бойынша талдау жасап, эмоциялар, кілт сөздер мен тақырыптарды анықтаңыз.
    4. Нәтижелерді қарап, эмоцияларды талдаңыз: мәтінде позитивті немесе негативті сипаттамалар бар ма?

    Мысал:

    Чат-боттармен жұмыс

    Чат-боттар — бұл автоматтандырылған жүйелер, олар адамдармен әңгімелесу үшін арнайы бағдарламалармен жасалған. Чат-боттар сұрақтарға жауап беруге, тапсырмаларды орындауға және кейбір жағдайларда пайдаланушылармен интерактивті түрде сөйлесуге арналған.

    Диалогтық жүйелер: Чат-боттар көбінесе диалогтық жүйелер болып табылады. Олар өзіне берілген сұрақтарға жауап беру үшін алдын ала дайындалған ережелер мен шаблондар арқылы жұмыс істейді.

    Intents (Намерения): Intents — бұл чат-ботқа арналған әрбір сұрақтың немесе әрекеттің мәні. Мысалы, "Қалайсың?" деген сұрақ "Табиғаттан сұрау" intent-іне жатуы мүмкін.

    Entities (Сущности): Entities — бұл мәліметтер, олар сұрақтың маңызды бөлігін құрайды. Мысалы, "Қазір қандай ауа-райы?" деген сұрақтың "ауа-райы" сөзі — entity болады.

    2. MNIST деректер жиынтығын жүктеп алыңыз:

      train_images = train_images / 255.0
      test_images = test_images / 255.0

      model = Sequential([
          Flatten(input_shape=(28, 28)),
          Dense(128, activation='relu'),

      model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
    6. Модельді оқыту: 
      model.fit(train_images, train_labels, epochs=5)
    7. Нәтижелерді бағалау: 
      model.evaluate(test_images, test_labels)

    Ойындардағы жасанды интеллект

    Жасанды интеллект (ЖИ) ойындарда кеңінен қолданылады. Ойындарда ИИ қолдану арқылы қарсыластарды басқару, олардың әрекеттерін ойыншының әрекетіне байланысты бейімдеу және жолдарды іздеу сияқты тапсырмаларды орындауға болады. Бұл ЖИ алгоритмдерінің ойынға әсерін түсінуге көмектеседі.

    Алгоритмдер: Видеоиграларда ЖИ кейбір әрекеттерді автоматты түрде орындау үшін алгоритмдер қолданады. Мысалы, A алгоритмі жолды табуға және басқарушының қозғалысын бағыттауға қолданылады. Бұл алгоритм кейбір «ауыртпалықты» қалдырмай, жолды ең тиімді түрде табуға мүмкіндік береді.

    Қолданылуы: ЖИ ойындарда қолданушыларға қарсы күресетін ойыншыларды, көмекші құралдарды, сондай-ақ ойыншының қадамдарын болжау үшін маңызды элемент болып табылады. Сонымен қатар, кейбір ойындарда ИИ ойыншылардың өміріне әсер етеді.

    Практикалық тапсырма:

    Бұл тапсырмада сіз Pygame кітапханасын пайдаланып, қарапайым ойын жасайсыз. Ойынның мақсаты — ЖИ-қарсыластың экранда қозғалып, кедергілерден аулақ болуымен бірге ойыншының жолын қуу.

    Қадамдар:

    1. Pygame кітапханасын орнату: 
      pip install pygame
    2. Ойынның бастапқы экранын жасау: 
      import pygame
      pygame.init()
      screen = pygame.display.set_mode((500, 500))
    3. Ойыншы мен жасанды интеллекттің суретін салу: 
      player = pygame.Rect(200, 200, 50, 50)
      enemy = pygame.Rect(100, 100, 50, 50)
      pygame.draw.rect(screen, (0, 128, 255), player)
      pygame.draw.rect(screen, (255, 0, 0), enemy)
    4. Ойыншының қозғалысын басқару: 
      keys = pygame.key.get_pressed()
      if keys[pygame.K_LEFT]: player.x -= 5
      if keys[pygame.K_RIGHT]: player.x += 5
      if keys[pygame.K_UP]: player.y -= 5
      if keys[pygame.K_DOWN]: player.y += 5
    5. ЖИ қозғалысын басқару: 
      if player.x < enemy.x: enemy.x -= 2
      if player.x > enemy.x: enemy.x += 2
      if player.y < enemy.y: enemy.y -= 2
      if player.y > enemy.y: enemy.y += 2
    6. Кедергілер мен шектеулерді енгізу: 
      if enemy.colliderect(player):
          print("Game Over!")
    7. Ойынның нәтижелерін шығару: 
      pygame.display.update()

    Мысал:

    import pygame
    pygame.init()
    screen = pygame.display.set_mode((500, 500))
    player = pygame.Rect(200, 200, 50, 50)
    enemy = pygame.Rect(100, 100, 50, 50)
    running = True
    while running:
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                running = False
        keys = pygame.key.get_pressed()
        if keys[pygame.K_LEFT]: player.x -= 5
        if keys[pygame.K_RIGHT]: player.x += 5
        if keys[pygame.K_UP]: player.y -= 5
        if keys[pygame.K_DOWN]: player.y += 5
        if player.x < enemy.x: enemy.x -= 2
        if player.x > enemy.x: enemy.x += 2
        if player.y < enemy.y: enemy.y -= 2
        if player.y > enemy.y: enemy.y += 2
        screen.fill((255, 255, 255))
        pygame.draw.rect(screen, (0, 128, 255), player)
        pygame.draw.rect(screen, (255, 0, 0), enemy)
        if enemy.colliderect(player):
            print("Game Over!")
        pygame.display.update()
    pygame.quit()
